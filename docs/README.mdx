**Hotelgest Unified** es un sistema de gestión hotelera (PMS) para hoteles, apartamentos turísticos y alojamientos. Gestiona reservas, facturación, housekeeping, comunicaciones e informes desde una única plataforma web y móvil.

## Acceso al sistema

<Callout type="info" title="Necesitas">
  1. Un **nombre de usuario** proporcionado por tu administrador  
  2. Una **contraseña** segura  
  3. Opcionalmente, una aplicación de **autenticación 2FA** (Google Authenticator o similar)  
</Callout>

Consulta [Inicio y Autenticación](/01-inicio-y-autenticacion) para más detalles.
