Hotelgest Unified se integra con múltiples servicios externos para automatizar procesos, cumplir la legislación y ampliar las funcionalidades del PMS.

Ver Marketplace

Autoridades españolas

Envío automático o manual de partes de viajeros: Ertzaintza (País Vasco), SES (estatal), Mossos d'Esquadra (Cataluña). Al check-in los datos se envían a la autoridad correspondiente. Estado en el informe Autoridades: Enviado, Error, Pendiente de reintento. Desde la reserva, Autoridades para enviar o reenviar manualmente.

Firma digital

Verifactu: Facturación electrónica y firma digital conforme a normativa española; código QR en facturas. TicketBai (País Vasco): Facturación obligatoria, firma y envío a Hacienda Foral.

Channel Manager (Channex)

Sincronización de disponibilidad y tarifas en tiempo real, recepción de reservas de OTAs, prevención de overbooking. OTAs: Booking.com, Expedia, Airbnb y otros. Configuración en Configuración > Channel Manager.

Cerraduras electrónicas (Doors)

Códigos PIN automáticos, llaves digitales al móvil, control de acceso por habitación y período, registro de accesos. Desde la reserva, Llaves: ver códigos, generar o desactivar.

TV (Yuvod) y CRM

Yuvod: pantalla de bienvenida en la TV, información del huésped, servicios desde la TV. CRM: Clientify y Pipedrive para sincronización de contactos y webhooks de reservas.

Marketplace

Catálogo de integraciones en el menú. Explorar por categoría, activar/desactivar, configurar, ver estado. Categorías: autoridades, firma digital, Channel Manager, cerraduras, TV, CRM, comunicación, pasarelas de pago.

Webhooks y API pública

Webhooks envían datos a URLs cuando ocurren eventos (reserva, check-in, pago, etc.). API REST: requiere API Key (Configuración > API Keys), cabecera x-api-key, documentación Swagger/OpenAPI para integraciones propias.
