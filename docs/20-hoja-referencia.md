# Hoja de referencia rápida

**Para qué sirve.** Esta página resume accesos directos y dónde está cada cosa en Hotelgest. Puede imprimirla o guardarla en PDF para tenerla a mano.

---

## Dónde entrar

- **Aplicación web:** [app.hotelgest.com](https://app.hotelgest.com)
- **Iniciar sesión:** [app.hotelgest.com/login](https://app.hotelgest.com/login)

---

## Dónde está cada cosa en el menú

| Qué quiere hacer           | Dónde ir en el programa      |
|----------------------------|------------------------------|
| Ver el resumen del día     | Panel de control (Dashboard) |
| Ver y gestionar reservas   | Planificación / Reservas      |
| Ver disponibilidad         | Planificación / Disponibilidad|
| Ver una reserva concreta   | Clic en la reserva en el calendario |
| Limpieza y estado habitaciones | Housekeeping             |
| Mensajes y avisos          | Mensajes / Notificaciones    |
| Ver actividad y cambios    | Actividad                    |
| Informes (reservas, facturas, etc.) | Informes / Reportes   |
| Emitir facturas y pagos    | Dentro de la ficha de la reserva, pestaña Facturación |
| Configurar el programa     | Ajustes / Configuración      |
| Ayuda y guías              | Academia / Ayuda             |

---

## Atajos útiles en el navegador

- **Ampliar la página (zoom):** Ctrl + "+" (en Windows) o Cmd + "+" (en Mac). Para reducir: Ctrl o Cmd + "-".
- **Actualizar la página:** F5 o el botón de actualizar del navegador (por si algo no se ve bien).
- **Ir atrás:** Botón "Atrás" del navegador o Alt + Flecha izquierda (Windows).

---

## Pasos muy breves: uso diario

1. **Entrar:** app.hotelgest.com/login → usuario → Continuar → contraseña → Iniciar sesión.
2. **Ver reservas de hoy:** Panel de control o Planificación → elegir el día.
3. **Abrir una reserva:** Planificación → clic en la reserva → se abre la ficha.
4. **Registrar un pago:** Ficha de la reserva → pestaña Facturación / Pagos → botón de añadir pago.
5. **Hacer check-in:** Ficha de la reserva → botón Check-in (o cambio de estado a "En curso").

---

## Si algo no funciona

- No puede iniciar sesión → [Resolución de problemas: inicio de sesión](/docs/18-resolucion-problemas#no-puedo-iniciar-sesión).
- No ve reservas → Comprobar fechas y filtros; actualizar la página.
- No se descarga un informe → Comprobar que el navegador permite descargas; esperar unos segundos; probar otro navegador.

Puede volver a intentarlo cuando quiera. Sus datos no se pierden.
