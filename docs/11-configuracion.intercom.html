El módulo de Configuración permite personalizar todos los aspectos de Hotelgest Unified. Se accede desde el menú Configuración (icono de engranaje).

Ir a Configuración

Perfil

Datos personales (nombre, apellidos, usuario, email), cambiar contraseña, autenticación de dos factores (2FA), sesiones activas y cerrar sesiones, consentimiento de datos.

Usuarios

Lista con usuario, nombre, email, rol y habitaciones asignadas. Acciones: añadir, editar, bloquear/desbloquear, compartir credenciales, eliminar, 2FA, reenviar email de verificación.

Roles

Crear roles con permisos por módulo: Dashboard, Calendario, Reservas, Housekeeping, Menú, Configuración. Editar, eliminar, exportar o crear desde plantilla.

Suscripción, API Keys y Webhooks

Suscripción: Plan actual, cambiar de plan, facturación (Chargebee). API Keys: Crear claves con descripción; la clave se muestra solo una vez. Webhooks: URL de destino, eventos (nueva reserva, modificación, cancelación, check-in, check-out, pago), cabeceras opcionales.

Auto-asignación de tareas y Comunicación

Reglas automáticas: nombre, condiciones, tareas creadas, recursos. Plantillas de WhatsApp: crear, editar, activar/desactivar, idiomas.

Políticas, Cuenta y Privacidad

Políticas de cancelación (Flexible, No reembolsable). Cuenta: tipo de propiedad, nombre, teléfono, email, web, moneda, logo, descripción, ubicación, galería. Privacidad: retención de datos (1–10 años) y datos afectados (RGPD).

Impuestos, Contabilidad y Habitaciones

Impuestos: IVA, impuestos adicionales, tasa turística (precio por adulto/niño, noches, recargo, tarifas por temporada). Contabilidad: Datos fiscales, prefijos de facturas/proformas/notas de crédito, impuestos, campos obligatorios, hora de cierre, pie de factura. Habitaciones: Tipologías con nombre, camas, baños, tamaño, ocupación, número de puerta, planta, orden, inventario, motor de reservas, fotos, coste de limpieza, tipo INE.

Reservas (configuración) y Channel Manager

Opciones de reservas: campos obligatorios, decimales, días de limpieza, horas check-in/check-out, asignación de habitación, check-out restrictivo, colores, emails automáticos, SMTP. Recepción Autónoma: ver doc de Recepción Automática. Channel Manager (Channex): inventario, reservas, mapeo de habitaciones, canales (Booking, Expedia, etc.).
