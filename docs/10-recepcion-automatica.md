# 10 - Recepción Automática

**URL recepción automática (pantalla de gestión):** [https://app.hotelgest.com/automatic-reception](https://app.hotelgest.com/automatic-reception)  
**Configuración:** [https://app.hotelgest.com/settings/autonomousReception](https://app.hotelgest.com/settings/autonomousReception)

La Recepción Automática (también llamada Recepción Autónoma) permite a los huéspedes realizar el check-in y check-out de forma autónoma sin necesidad de intervención del personal de recepción.

---

## Descripción General

El sistema de Recepción Automática ofrece una **web app** que se envía al huésped para que pueda:
- Completar el registro de viajeros antes de llegar
- Escanear su documento de identidad
- Firmar digitalmente el registro
- Recibir instrucciones de acceso (código PIN, llaves digitales)
- Realizar el check-out

---

## Flujo del Huésped

### 1. Antes de la llegada
1. El huésped recibe un **email o WhatsApp** con el enlace a la web app
2. Abre el enlace en su navegador
3. Completa sus **datos personales**
4. **Escanea su documento de identidad** con la cámara del móvil
5. **Firma digitalmente** el registro de viajeros
6. Recibe la confirmación

### 2. Llegada
- Si el check-in online está completado, el huésped recibe:
  - **Código PIN** para acceder al alojamiento (si hay cerraduras integradas)
  - Instrucciones de acceso
  - Información del alojamiento (WiFi, normas, etc.)

### 3. Check-out
- El huésped puede completar el check-out desde la web app
- Se cierra automáticamente la estancia

---

## Configuración

### Acceso
Desde **Configuración > Recepción Autónoma** puedes configurar:

### Opciones generales
| Opción | Descripción |
|--------|-------------|
| **Web App** | Activar/desactivar la web app de recepción automática |
| **Check-in online estricto** | Si se requiere que todos los huéspedes completen el check-in online |
| **Código PIN** | Activar/desactivar la generación de códigos PIN |
| **Widget del tiempo** | Mostrar información meteorológica en la web app |
| **Hora de generación del PIN** | A qué hora se genera y envía el código PIN |
| **Check-in** | Opciones específicas del proceso de check-in |
| **Check-out** | Opciones específicas del proceso de check-out |

---

## Enviar Web App al Huésped

Desde el detalle de la reserva:
1. Haz clic en **Web App** en la cabecera de acciones
2. El sistema genera un enlace único para esa reserva
3. Se envía al huésped por email o WhatsApp
4. El huésped puede completar el registro desde cualquier dispositivo

---

## Widget del Tiempo

Si está activado, la web app muestra al huésped:
- Temperatura actual en la ubicación del alojamiento
- Previsión meteorológica para los próximos días

---

## Estado del Check-in Online

Desde la reserva puedes ver el estado del check-in online:
- **Pendiente**: El huésped aún no ha iniciado el proceso
- **En espera**: El huésped ha iniciado pero no ha completado
- **Completado**: El huésped ha finalizado el registro online

---

## Ventajas

- **Reduce colas** en recepción
- **Ahorra tiempo** al personal
- **Mejora la experiencia** del huésped
- **Cumplimiento legal**: Los datos se envían automáticamente a las autoridades
- **Disponible 24/7**: Los huéspedes pueden registrarse a cualquier hora
- **Sin contacto**: Proceso completamente digital
